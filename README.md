# Overview

This was developed as part of the **Software Engineering (ES)** course at IST-UL.

The objective was to familiarize us how to work as a team in a git repository.

The project in itself was to extend the HumanaEthica app by adding a Shift class (Activity is divided into Shifts). There were 2 deliverables: sprint-1 and sprint-2.

More details can be found in the README.md of the sprint directories.

# sprint-1

This part required to implement the Shift into the backend, creating the class, services and web services. Created new tests and ensured previous ones still passed.

Used
- JPA for data persistence
- Spock to create tests
- Spring-boot to create REST services

# sprint-2

In this part, we used the backend code (with the Shift implemented), given by the teaching staff (to ensure correctness), and created the Shift logic for the frontend.

Used
- Vue.js to create the UI
- TypeScript for models and services
- JavaScript to create e2e tests
- Cypress to execute the tests
